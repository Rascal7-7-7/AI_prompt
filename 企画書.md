# 企画書 — プロンプトエンジニアリング学習アプリ

## プロジェクト概要

| 項目 | 内容 |
|---|---|
| プロジェクト名 | 未定（仮称：PromptLab） |
| 種別 | Webアプリ（Next.js フルスタック） |
| 制作目的 | 個人学習 + ポートフォリオ |
| 想定公開形態 | Vercel（公開URL） |
| 制作者 | 専門学校生（2年生） |
| 制作期限 | なし（クオリティ重視） |

---

## 解決する課題

IT系専門学校に通う学生の多くが「AIは知っているが、使い方がわからない」状態にある。

特に以下の点でつまずきやすい：

- プロンプトを「どう書けばいいか」がわからない
- Claude と ChatGPT の使い分けがわからない
- Claude Code や AGENTS.md など開発者向けツールの存在を知らない
- 画像生成・動画生成プロンプトの書き方がわからない
- 「知識として知っている」と「実際に使える」の間に大きなギャップがある

---

## 主要機能

### コア機能

| 機能 | 概要 |
|---|---|
| チャプター別学習 | 7チャプター構成。サイドバーから章を選択して学習 |
| プロンプト穴埋め問題 | 4択でプロンプトの空欄を埋める形式 |
| Before / After 比較 | 悪いプロンプトと改善後を並べて視覚的に学習 |
| インタラクティブサンドボックス | テキスト系はClaudeへの実際のAPI問い合わせを体験 |
| 画像生成サンプル展示 | プロンプト + 公式サンプル画像 / 自作サンプルをセット表示 |
| チャプタークイズ | 章末理解度チェック。合格で次章解放 |
| 進捗管理 | 完了チャプター・クイズスコアを記録 |
| Claude vs ChatGPT 比較 | 同タスクでの2モデル出力を並列表示 |

### 補助機能

| 機能 | 概要 |
|---|---|
| ダークモード対応 | ユーザー設定で切り替え |
| レスポンシブ対応 | モバイル・タブレット・PCすべてに対応 |
| 進捗リセット | 学習をやり直せる |

## カリキュラム構成（7チャプター）

### Chapter 1：AI基礎
- LLM（大規模言語モデル）とは何か
- Claude と ChatGPT の違い・特徴比較
- どのツールをいつ使うべきか

### Chapter 2：プロンプト基礎
- プロンプトの構造（役割・文脈・指示・出力形式）
- 曖昧な指示 vs 具体的な指示（Before / After）
- XMLタグを使った構造化
- ChatGPT 黄金フォーマット（ゴール / 出力形式 / 注意点 / 背景情報）

### Chapter 3：Claude 活用
- Claude Code — コーディング支援プロンプトの書き方
- System Prompt / カスタム指示の設計
- Artifacts 活用（コード・ドキュメント生成）
- Claude in Excel / PowerPoint の活用

### Chapter 4：ChatGPT 活用
- 文章・ライティング支援プロンプト
- Codex / GPT-4o — コード生成プロンプト
- データ分析・表整理プロンプト
- Custom Instructions の設計

### Chapter 5：画像・動画生成
- GPT-Image-1.5（旧DALL·E 3）— 画像生成プロンプト技法
- スタイル指定・構図指定・照明指定の書き方
- Sora — 動画生成プロンプトの構造
- 英語プロンプト推奨の理由と翻訳ツール活用

### Chapter 6：設定ドキュメント（開発者向け）
- CLAUDE.md とは何か・なぜ書くのか
- AGENTS.md の記述パターン（役割 / ルール / 禁止事項）
- loop.md / design.md の設計思想
- 実例：このアプリのAGENTS.mdを教材として使用

### Chapter 7：実践・比較演習
- 同一タスクで Claude vs ChatGPT を比較
- プロンプト改善チャレンジ（採点あり）
- 実務シナリオ別おすすめプロンプト集
- 次のステップ（公式ドキュメント・コミュニティ）

## コンテンツ素材の方針

### プロンプトサンプル

- note の RSS フィード + 手動キュレーションでパターンを収集
- 収集したパターンを参考に、制作者が独自に再現・検証したものを掲載
- 出典が明確なものは「参考：〇〇さんの手法を元に作成」と明記

### 画像サンプル（Chapter 5）

- OpenAI 公式サイト・Sora 公式ページの公開サンプルを引用（出典明示）
- 著作権法第32条（引用）に基づき掲載
- 可能であれば制作者が実際に生成した画像も追加
